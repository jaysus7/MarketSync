18. Chrome Extension — Single Purpose & Limited Use

The MarketSync Chrome Extension is built with a single, narrow purpose: to automate inventory cross-listing by parsing active product details from user-selected source inventory tabs and localizing/formatting that data to sync into destination marketplace forms. Data parsed dynamically from web pages via the activeTab and scripting APIs is ephemeral and handled in short-lived runtime memory; local preferences are stored securely within your sandboxed Chrome profile via chrome.storage.

In accordance with the Chrome Web Store Developer Program Policies, MarketSync certifies the following handling mandates:

  • No Sale or Transfer: We do not sell or transfer user data to third parties, outside of the approved operational use cases explicitly required to fulfill the syncing action.
  • Purpose Limitation: We do not use or transfer user data for any purposes unrelated to the core single purpose of cross-listing automation.
  • No Credit Scoring: We do not use or transfer user data to determine consumer creditworthiness or for lending evaluation purposes.
